# 后续功能规划

以下功能是后续版本的候选方向，当前版本尚未实现。

## 任务管理

- 在聊天中查询下载进度、队列位置和当前任务状态。
- 支持失败图片重试、断点续传和重启后的任务恢复。

## 缓存管理

- 在管理页面中查询和删除单个缓存文件。
- 按 JM 号清理缓存。
- 仅下载并合并新增章节。

## 权限与限流

- 群聊白名单和管理员专用模式。
- 针对群聊和私聊分别设置策略。

## 下载与发送

- 最大文件体积预估与自适应压缩质量。
- 超大文件分卷压缩或分段发送。
- 收藏订阅和新章节通知。

## 管理与观测

- WebUI 实时进度流和历史趋势图。
- 结构化指标导出与外部监控接入。
