# 更新日志

本项目遵循语义化版本，并从 `v1.0.0` 开始记录公开版本变更。

## v1.1.0 - 2026-07-13

- 增加用户级请求速率、每日下载次数和每日发送流量配额。
- 增加域名健康检查，自动优先使用可用且延迟较低的镜像。
- 增加可选 JPEG 重编码和最大宽度限制，以减小 PDF 体积。
- 增加 AstrBot 原生管理 Page，展示任务、用量、域名、缓存、事件与失败原因。
- 增加配置脱敏/完整导出、导入前验证和运行时热重载。
- 增加 AstrBot 管理员及插件 QQ 管理员任务取消命令。
- 增加缓存空间配额、按最后使用时间自动过期和手动清理。
- 使用 SQLite 持久化用量、结构化事件、域名状态和运行指标。

## v1.0.0 - 2026-07-13

- 支持群聊和私聊通过 `/本子 <jm号>` 下载漫画。
- 支持章节选择、独立章节上限和严格图片页数上限。
- 使用精确图片清单生成 PDF，修复跨章节重复打包和限制失效问题。
- 使用 AES-256 加密 ZIP，并通过实际解密验证密码与缓存。
- 支持缓存、重复任务合并、发送冷却、全局队列和单会话限流。
- 使用合并转发展示长章节列表，并在有效选择后尝试撤回用户消息。
- 增加配置校验、敏感错误脱敏、自动化测试、CI 和 7-Zip 发布脚本。
